# 💰 Expense Tracker CLI (Python)

A menu-driven command-line expense tracker built using Python and CSV file handling.

## 🚀 Features
- Add Expense
- View All Expenses
- Search by Category
- Monthly Total Calculation
- Delete Expense by ID
- Auto CSV File Creation

## 🛠 Tech Used
- Python 3
- CSV Module
- Datetime Module

## 📂 File Structure
- task4.py
- expenses.csv (auto-generated)

## ▶️ How to Run

```bash
python expense_tracker.py
